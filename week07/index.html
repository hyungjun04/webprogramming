🚀 주요 기능 가이드
백스페이스 (←)

입력한 숫자의 맨 오른쪽 숫자 하나를 지웁니다. 숫자가 하나만 남았을 때 누르면 0으로 초기화됩니다.

팩토리얼 (x!)

1부터 입력한 양의 정수까지 모두 곱합니다. (예: 5! = 120). 음수 입력 시 Error가 표시됩니다.

역수 (1/x)

1을 현재 숫자로 나눈 값을 구합니다. 분모가 0인 경우 계산이 불가능하여 에러가 발생합니다.

상용로그 (log)

밑이 10인 로그 값을 계산합니다. 로그 정의에 따라 0 이하의 숫자는 에러 처리됩니다.
