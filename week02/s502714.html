●웹프로그래밍(502714)

Web Programming

이 교과목은 웹개발의 기초 원리를 이해하고 HTML,CSS,JavaScript를 활용한 웹 페이지 구축을 학습합니다. 학생들은 웹 표준, 레이아웃, 스타일링, 상호작용, 그리고 기본 알고리즘에 대한 이해를 키우며, 프로젝트를 통해 실제 웹 애플리케이션을 개발하는 경험을 얻습니다.

This course covers fundamental principles of web development, focusing on understanding HTML,CSS,and JavaScript for building web pages. Students will gain knowledge in web standards, layout, styling, interactivity, and basic algorithms while developing real web applications through projects.

웹프로그래밍을 가장 먼저 선정한 이유는 교수님이 하나하나 친절하게 잘 가르쳐주시고, 내가 만든 표가 망가지면 화가 나기도 하지만, 이루면 뿌듯함이 있기 때문이다.
